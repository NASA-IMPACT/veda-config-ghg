  <Prose>
    The EPA Anthropogenic Methane Emissions (Express Extension) data product contains a gridded inventory of annual anthropogenic methane emissions for 2012-2020, consistent with the EPA’s Inventory of U.S. Greenhouse Gas Emissions and Sinks (GHGI) published in 2022. The extension dataset was developed to incorporate updates that the EPA has made to the U.S. GHGI since the publication of the 2020 U.S. GHGI Report. It uses the same gridding methodology as the main gridded GHGI dataset but emissions are scaled to be consistent with those from the 2022 U.S. GHGI Report, allowing for comparison with more recent methane emissions estimates. The data are provided over the contiguous United States on a 0.1 x 0.1 degree grid for 27 source categories relating to agriculture, natural gas systems, petroleum systems, waste, and coal mines among others. The data have been converted from NetCDF to Cloud-Optimized GeoTIFF (COG) for use in the U.S. GHG Center, enabling robust comparisons of emissions and their trends.
   
    - **Temporal Extent:** 2012 - 2020    
    - **Temporal Resolution:** Annual    
    - **Spatial Extent:** Contiguous United States  
    - **Spatial Resolution:** 0.1° x 0.1°    
    - **Data Units:** Molecules of methane per square centimeter per second (Molecules CH₄ cm²/s)    
    - **Data type:** Regulatory
    - **Data Latency:** N/A

    **Scientific Details:** The annual files include a ‘Grid_cell_area’ variable to help users convert the emission fluxes to emission mass totals. The variable also includes a time dimension to make it suitable (COARDS-compliant) for use in atmospheric models. In addition to annual emission flux files, the main gridded dataset also includes monthly source-specific methane emission scaling factors for sources with strong interannual variability. For these select sources, the scaling factors can be multiplied with the data in the annual emission files to obtain monthly methane emission fluxes. Monthly data is not included in the GHG Center.

    ## Dataset Accuracy
    Using inverse analysis, uncertainty and error variances are estimated by comparing source- specific 2012 gridded estimates to those from an external detailed bottom-up inventory for the Barnett region in Texas that was compiled in 2015 and matches atmospheric measurements. For more information on uncertainty estimates, refer to the Maasakkers et al. (2023) paper.	 
    
    ## Disclaimer
    All data provided in the U.S. GHG Center has been transformed from the original format (NetCDF) into Cloud Optimized GeoTIFF ([COG] (https://www.cogeo.org/)). Careful quality checks are used to ensure data transformation has been performed correctly. 

    Note that this dataset does not currently include emissions from the Land Use, Land Use Change, and Forestry (LULUCF) category of the national U.S. GHGI. Methane emissions from these sources include but are not limited to emissions from fires and flooded lands. 

    The EPA Anthropogenic Methane Emissions (Express Extension) data is considered an extension of the main gridded GHGI dataset that’s consistent with the 2020 U.S. GHGI Report. The express extension dataset is not a full update to the main gridded GHGI dataset, and instead uses annual source-specific spatial patterns from 2012-2018 from the main dataset to quickly incorporate more recent national methane emission estimates. Regardless of application, users should not combine emissions data from the main gridded dataset with the express extension. Additionally, for years after 2018, monthly scaling factors should only be applied to the express extension for manure management, rice cultivation, and field burning of agricultural residue emissions. For more information on the differences between the main gridded GHGI dataset and the express extension, visit the Gridded EPA U.S. Anthropogenic Methane Greenhouse Gas Inventory.
